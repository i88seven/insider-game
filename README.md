# insider-game

## Build Setup

```bash
# install dependencies
$ yarn install

$ cp .env.example .env
# Set LIFF_ID etc..
```

## Build local server

Build LIFF app. Set `https://localhost:3000` on endpoint URL.

```bash
# serve with hot reload at https://localhost:3000
$ yarn dev
```
